# EstateFlow

A full-stack Real Estate CRM platform built using Spring Boot and React.

## Features

- JWT Authentication
- Role-Based Access
- Lead Management
- Property Management
- Dashboard Analytics
- Search and Filtering

## Tech Stack

### Backend
- Java
- Spring Boot
- Spring Security
- JWT
- Hibernate
- MySQL

### Frontend
- React
- Axios
- React Router
- Chart.js
